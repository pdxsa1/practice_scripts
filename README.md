# Interview Practice Scripts

A collection of Python scripts built to practice automation and backend concepts.

## Scripts

### filter_csv.py
Reads employee data from a CSV file and filters results based on salary threshold.

### rename_files.py
Automates file renaming by prepending today's date to all files in a directory.

### api_call.py
Fetches live Bitcoin price data from the Coinbase public API and parses the response.

## Skills Demonstrated
- File I/O and CSV parsing
- File system automation with the os module
- REST API calls and JSON parsing
- Python datetime handling
